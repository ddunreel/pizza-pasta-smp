⚙ OCI Setup

Add these secrets to your GitHub repo (Settings > Secrets > Actions):

OCI_TENANCY_OCID - Your tenancy OCID
OCI_USER_OCID - Your OCI user OCID
OCI_FINGERPRINT - API key fingerprint
OCI_PRIVATE_KEY - Full PEM private key content
OCI_REGION - e.g. eu-frankfurt-1
OCI_COMPARTMENT_OCID - Compartment for VMs

Generate an API key in OCI Console > Profile > API Keys > Add API Key.
